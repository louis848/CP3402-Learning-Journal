# Week 4
## Learning Activities and Resources
I reviewed the CP3402 material on development and deployment environments. I studied local environments, staging environments, and production environments, as well as Vagrant, Docker, GitHub, and Linux file permissions.
## Estimated Time
About 3 hours.
## Content Insights
I learned that websites shouldn’t be developed directly on live production servers because mistakes could affect real users. Local environments are for development, while staging environments allow testing before deployment.
## Career/Employment/Learning Insights
This topic made me realize that a good workflow is really important for teamwork and reliable deployment. In future projects, I should use version control, test changes before release, and make backups before updating the website or database.
