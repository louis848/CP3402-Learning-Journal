# Week 5
## Learning Activities and Resources
I reviewed the CP3402 materials on WordPress child themes. I learned that child themes allow you to safely customize a parent theme without losing changes after updates.
## Estimated Study Time
About 2 hours.
## Content Insights
I found out that child themes use their own `style.css` and `functions.php` files while inheriting the functionality of the parent theme. They only override specific template files when needed, making customization safer and easier to maintain.
## Career/Employment/Learning Insights
This topic made me realize the importance of good development practices instead of directly modifying existing files. Using child themes, version control, browser tools, and coding standards makes website development more reliable and professional.
